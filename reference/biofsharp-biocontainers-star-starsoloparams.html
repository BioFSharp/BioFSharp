CBposition(string)
Signature: string

position of Cell Barcode(s) on the barcode read. Presently only works with --soloType CBUMIComplex, and barcodes are assumed to be on Read2. Format for each barcode: startAnchorstartDistanceendAnchorendDistance start(end)Anchor defines the anchor base for the CB: 0: read start; 1: read end; 2: adapter start; 3: adapter end start(end)Distance is the distance from the CB start(end) to the Anchor base String for different barcodes are separated by space. Example: inDrop (Zilionis et al, Nat. Protocols, 2017): --soloCBposition 002-1 313_8
