MatchBetweenRunsOptions

Namespace: BioFSharp.BioContainers
Parent Module: MoFF

Those options run the MBR module, which takes a list of features of interest for a given run, and then matches the corresponding features in other runs. The output will be stored in a subfolder ('mbroutput') inside the specified input folder. The MBR module will consider all the .txt files present in the specified input folder as replicates (to select specific files or different extension, please refer to the example below). The files in samplefolder/mbr_output will be identical to the input files, but they will have an additional field ('matched') that specifies which peptides have match (1) or not (0). The MBR algorithm also produces a log file in the provided input directory.
