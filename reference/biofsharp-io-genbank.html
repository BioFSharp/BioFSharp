GenBank

Namespace: BioFsharp.IO

functions for reading and writing GenBank(.gb) files

Nested types and modules

TypeDescription
BaseSpanInformation

Represents the possible formats of a feature base span
BaseSpanRange

Distincts BaseSpans by joins,complements or single ranges
Feature

Represents a single feature from the FEATURES section of a GenBank file. Features can contain Information about genes and gene products, as well as regions of biological significance reported in the sequence
FeatureQualifier

Represents a single feature Qualifier and its value from the FEATURES section of a Genbank file. Features can contain Information about genes and gene products, as well as regions of biological significance reported in the sequence
GenBankItem

Represents any Item a GenBank file can contain as a union case. The result of parsing a genBank file will be a dictionary containing this type.
Origin

Represents the origin part of a GeneBank file.
Reference

Represents a single reference from the REFERENCE section of a GenBank file. A Reference represents publications by the authors of the sequence that discuss the data reported in the record.

Returns a list of features containing the input BaseSpanRange. if matchExact = true, returns only the features with exactly matching baseSpans. if matchExact = false, returns all distinct features that are contain any of the BaseSpans present in the input BaseSpanRange (still exactly matching the individual BaseSpan exactly).
