Namespace: BioFSharp
Parent Module: Isotopes

Contains the stable isotopes of biologically relevant chemical elements.

Note that the name convention of isotopes e.g. 15N cannot be satisfied here, as name bindings starting with numbers are not allowed.

Therefore, numbers and element symbols are switched, e.g. 15N becoming N15
